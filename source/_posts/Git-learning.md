# Git_learning

# Git基本操作

记录目前已经用到过的git操作，后续如果有发生新的操作再进行修改

**创建仓库**

```bash
git init
git clone
```

**提交和修改**

```bash
git add
git status
git diff
git commit
```

**日志**

```bash
git log
#git blame <file>
```

**远程操作**

```bash
git remote
git fetch
git pull
git push
```

**分支**

```bash
git branch -a -v -av -d -b
git checkout
git merge
```
